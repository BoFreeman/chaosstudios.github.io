# Chaos Studios Deployment Kit

## Description
This is the official homepage for Chaos Studios: *Antisystem Satiricum*.

## Structure

- **index.html**: The main landing page featuring posters and call-to-action.
- **/assets/posters/**: Place final poster images here. Example:
  - nobody-kills-like-me.png
  - tariffs-are-beautiful.png
- **/assets/fonts/**: Optional fonts directory.
- **/scripts/**: Optional JS scripts (e.g. email capture or interactions).

## Hosting

Enable GitHub Pages for this repo under Settings > Pages > Source: `main` branch > root folder.

Live site: https://yourusername.github.io/chaos-studios-site/
